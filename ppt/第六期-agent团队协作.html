// 管理器  ·  团队配置与状态机
团队配置 · 状态机
固定队友需要
身份和状态
`.team/config.json` 是团队花名册，记录名字、角色和状态，尤其标记重启后的 offline。
人
name / role
alice 是 coder，bob 是 reviewer
态
working / idle
线程活着：正在工作或等待消息
离
offline / shutdown
重启后旧线程消失；需重新 spawn 才会处理 inbox
.team/config.json
{
  "team_name": "default",
  "members": [
    {
      "name": "alice",
      "role": "coder",
      "status": "idle"
    }
  ]
}

启动时：idle / working → offline
名字还在，信箱还在，但人没有上线。
