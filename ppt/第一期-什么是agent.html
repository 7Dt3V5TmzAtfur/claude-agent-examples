// slide 05  —  Agent 四要素
CONCEPT · FOUR ELEMENTS
Agent 四要素
🧭
规划 · Planning
BRAIN · 大脑
分解任务、制定步骤、推理决策，是 Agent 的思考中枢。
💾
记忆 · Memory
EXPERIENCE · 经验库
短期上下文 + 长期知识存储，让 Agent 能跨轮次记住信息。
🔧
工具 · Tools
HANDS · 手脚
文件读写、代码执行、API 调用，是 Agent 改变世界的方式。
👁️
感知 · Perception
SENSES · 五官
读取文本、图像、代码和工具返回结果，感知当前环境状态。
// slide 06  —  Agent 核心循环
CONCEPT · AGENT LOOP
Agent
核心循环
LLM 不断思考、选择工具、执行、观察结果，直到任务完成。
①
思考 · Think
llm.think(messages)
②
选择工具 · Choose
response.tool_calls
③
执行工具 · Execute
execute_tool(tool)
↩
记录结果，继续
messages.append(result)
agent_loop.py
1def agent_loop(user_input):
2  messages = [user_input]
3
4  while True:
5    # 1. 思考
6    response = llm.think(messages)
7
8    # 2. 判断是否完成
9    if response.done:
10      return response.answer
11
12    # 3. 执行工具
13    result = execute_tool(response.tool)
14
15    # 4. 记录结果
16    messages.append(result)
