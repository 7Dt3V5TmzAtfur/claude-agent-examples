CONCEPT 05 · TOOL USE
Tool Use
工具调用
告诉模型「你有哪些工具」，模型决定何时调用，代码执行并把结果返回，完成推理闭环。
LLM
① 模型决策
stop_reason == "tool_use"
run
② 代码执行工具
subprocess.run(command)
res
③ 返回 tool_result
{"type": "tool_result", ...}
LLM
④ 最终回答
stop_reason == "end_turn"

KEY
渐进式披露的本质是推迟信息注入——
System Prompt 只声明"存在感"，完整知识在真正被需要的那一刻才进入上下文，避免无效 token 消耗，同时保持无限可扩展。
